### 1.使用源代码在 Windows 上部署 PostgreSQL

https://www.youtube.com/watch?v=-BJmuZT5IPE

### 2.运行 PostgreSQL

- 启动

```
"pg_ctl" -D "D:\pgsql\pgdata95" start
```

- 登录

```
// psql --username=postgres
psql -d postgres
```

### 3.测试

- 在项目根目录`D:\JXMYJ\PostgreSQL\build\REL9_5_0`下运行

```
make installcheck
```

### 4.重新编译

- 在 MSYS2 MSYS 中，项目根目录`D:\JXMYJ\PostgreSQL\build\REL9_5_0`下运行

```
make clean
make
make install
```

- 在 dist 文件复制处 `D:\JXMYJ\REL9_5_0\bin` 下运行

```
initdb --pgdata=D:\pgsql\pgdata95 --username=JXMYJ --auth=trust
```

- 在 cmd 中运行

```
pg_ctl restart -D D:\pgsql\pgdata95
```

- 从头开始编译，在 MSYS2 MSYS 中，项目根目录`D:\JXMYJ\PostgreSQL\build\REL9_5_0`下运行

```
rel=REL9_5_0
source=/D/JXMYJ/PostgreSQL/source/
dist=/D/JXMYJ/PostgreSQL/dist/$rel
$source/configure --build=x86_64 --host=x86_64-w64-mingw32 --target=x86_64 --prefix=$dist $$ make $$ make install
```



```
"pg_ctl" stop -D "D:\pgsql\pgdata95"
make clean
make uninstall
make
make install
移动文件位置(D:\JXMYJ\PostgreSQL\dist\REL9_5_0 -> D:\JXMYJ\REL9_5_0)
删除原来的数据文件(D:\pgsql\pgdata95)
initdb --pgdata=D:\pgsql\pgdata95 --username=JXMYJ --auth=trust
"pg_ctl" -D "D:\pgsql\pgdata95" start
```
